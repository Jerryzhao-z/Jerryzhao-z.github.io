xgboost

2021-05-20 2 min read

xgboost是陈天奇在14年起开源的一个gradient tree boosting算法库，至今在很多工业场景下都是最优方案。

理论基础

xgboost全名是Extreme Gradient Boosting, 相关论文是XGBoost: A Scalable Tree Boosting System。从命名中就包含了以下几个递进的理论基础：

  • boosting 一类串行集成学习算法
  • gradient boosting 以梯度下降为优化方法的boosting算法
  • decision tree 决策树是一种监督学习算法，通过输入特征从树根开始进行if/then判断分叉，到达叶节点获得模型输出。
  • gradient tree boosting 以决策树为learner的gradient boosting算法

xgboost的优化

xgboost在决策树构建、工程化两个方面做了优化：

  • 在决策树构建方面：
    • 分裂点搜索
      • 名为 weighted quantile sketch的近似算法
      • 针对稀疏特征的优化
  • 工程设计
    • 特征统计阶段并行化
    • CPU空间局部性优化
    • IO优化

DART

DART给gradient tree boosting引入了dropout机制。

LightGBM

其实xgboost的训练速度已经很不错了，但是微软开源的gradient boosting库LightGBM，在训练速度和内存使用上做了优化，在一些公开数据集上能实现10倍以上的加速。
